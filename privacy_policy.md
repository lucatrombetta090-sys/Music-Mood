# Privacy Policy — Music-Mood

Data di efficacia: 7 luglio 2026

Music-Mood è un'app Android progettata per analizzare la libreria musicale locale dell'utente e generare classificazioni musicali per mood, statistiche personali e report settimanali. La protezione dei dati dell'utente è un principio fondamentale dell'app.

Questa informativa descrive quali dati possono essere trattati dall'app, per quali finalità, dove vengono conservati e quali opzioni ha l'utente.

## 2. Dati trattati dall'app

Music-Mood può trattare le seguenti categorie di dati, in base alle funzionalità utilizzate dall'utente.

### 2.1 Libreria musicale locale

Per funzionare, Music-Mood legge informazioni relative ai brani presenti sul dispositivo, tra cui:

- titolo del brano;
- artista;
- album;
- durata;
- eventuale genere;
- eventuale anno;
- percorso/cartella del file audio;
- eventuale copertina disponibile localmente;
- identificativo locale del brano.

Questi dati sono usati per mostrare la libreria musicale, organizzare i brani, filtrare e ordinare i contenuti, generare statistiche e preparare i report musicali.

### 2.2 Analisi mood dei brani

Music-Mood può analizzare i brani locali per attribuire un mood musicale, ad esempio:

- Positivo;
- Energico;
- Rilassato;
- Malinconico;
- Romantico;
- Aggressivo;
- Nostalgico;
- Concentrazione;
- Festivo.

L'app può inoltre salvare informazioni derivate dall'analisi, come:

- mood stimato;
- livello di confidenza;
- valenza;
- arousal;
- BPM stimato;
- tonalità musicale;
- eventuale mood modificato manualmente dall'utente.

Questi dati sono usati esclusivamente per funzionalità interne dell'app, come classificazione, ordinamento, statistiche e report.

### 2.3 Eventi di ascolto e Weekly Report

Music-Mood può registrare localmente eventi di ascolto e usarli per generare report settimanali. I report possono includere:

- numero di brani ascoltati;
- mood dominante;
- distribuzione percentuale dei mood;
- confronto con report precedenti;
- dati aggregati della settimana corrente o passata.

I report sono salvati localmente sul dispositivo.

### 2.4 Integrazione opzionale con Last.fm

Music-Mood può includere un'integrazione opzionale con Last.fm.

Se l'utente configura Last.fm, l'app può salvare localmente:

- username Last.fm;
- API key Last.fm inserita dall'utente;
- riepilogo locale degli ultimi dati Last.fm importati;
- statistiche aggregate Last.fm mostrate nel Weekly Report.

Quando l'utente preme il pulsante "Aggiorna Last.fm", Music-Mood può chiamare l'API Last.fm per recuperare ascolti recenti o ascolti relativi al periodo del report. I dati ottenuti possono includere:

- titolo del brano;
- artista;
- album;
- orario di ascolto, se disponibile;
- URL Last.fm del brano, se disponibile;
- URL immagine/copertina, se disponibile;
- indicazione di ascolto in corso, se disponibile.

L'integrazione Last.fm è opzionale. L'app non dovrebbe avviare automaticamente il refresh Last.fm all'apertura della schermata Weekly Report: l'aggiornamento viene eseguito dall'utente tramite pulsante dedicato.

### 2.5 Copertine e contenuti esterni

Music-Mood può visualizzare copertine dei brani quando disponibili localmente o tramite servizi esterni usati dall'app. Le copertine sono usate solo per migliorare l'esperienza visiva nella libreria, nel player e nel mini-player.

## 3. Finalità del trattamento

I dati sono trattati per:

- mostrare e organizzare la libreria musicale locale;
- classificare i brani per mood;
- permettere all'utente di correggere o personalizzare il mood;
- generare statistiche musicali;
- generare Weekly Mood Report;
- integrare facoltativamente dati Last.fm;
- migliorare l'esperienza utente tramite copertine, ordinamenti e filtri;
- consentire la condivisione manuale del report, se l'utente sceglie di farlo.

## 4. Conservazione dei dati

I dati dell'app sono conservati principalmente sul dispositivo dell'utente.

Possono essere conservati localmente:

- analisi mood dei brani;
- preferenze utente;
- configurazione Last.fm;
- cache del riepilogo Last.fm;
- report settimanali;
- eventi di ascolto;
- eventuali override manuali dei mood.

I dati rimangono sul dispositivo fino a quando l'utente:

- elimina i dati dell'app;
- disinstalla l'app;
- cancella le configurazioni o le analisi dall'app, se la funzionalità è disponibile;
- rimuove manualmente le credenziali Last.fm, se la funzionalità è disponibile.

## 5. Condivisione dei dati

Music-Mood non vende i dati personali dell'utente.

Nella configurazione attuale, l'app non usa un backend proprietario per elaborare i dati musicali dell'utente. Alcune funzionalità facoltative possono però comportare richieste verso servizi esterni:

- Last.fm, solo se l'utente configura l'integrazione e avvia manualmente l'aggiornamento;
- eventuali servizi esterni usati per recuperare metadati o copertine, se previsti dalla versione installata.

Quando l'utente usa Last.fm, il trattamento dei dati da parte di Last.fm è regolato anche dalle policy e dai termini di Last.fm.

## 6. API key Last.fm

L'API key Last.fm inserita dall'utente viene usata per effettuare chiamate verso l'API Last.fm.

L'utente è responsabile dell'uso della propria API key secondo i termini applicabili di Last.fm.

Per usi commerciali, di ricerca o accademici dell'API Last.fm, è opportuno verificare preventivamente le condizioni ufficiali di Last.fm.

## 7. Sicurezza

Music-Mood adotta un approccio di minimizzazione dei dati: i dati vengono usati per le funzionalità musicali dell'app e, ove possibile, conservati localmente.

Le comunicazioni verso servizi esterni, quando presenti, devono avvenire tramite connessioni sicure supportate dal servizio remoto.

Nessun sistema può garantire sicurezza assoluta, ma l'app è progettata per limitare la trasmissione dei dati allo stretto necessario per le funzionalità richieste dall'utente.

## 8. Permessi Android

Music-Mood può richiedere permessi necessari per accedere ai file audio o alla libreria multimediale del dispositivo, in base alla versione Android e alle funzionalità utilizzate.

I permessi richiesti sono usati per:

- leggere i brani presenti sul dispositivo;
- mostrare la libreria musicale;
- riprodurre musica;
- analizzare i brani;
- generare statistiche e report.

## 9. Condivisione manuale del Weekly Report

L'utente può scegliere di condividere manualmente un'immagine o un riepilogo del Weekly Mood Report usando le funzionalità di condivisione del dispositivo.

La condivisione avviene solo dopo un'azione esplicita dell'utente.

## 10. Minori

Music-Mood non è progettata specificamente per bambini e non intende raccogliere consapevolmente dati personali di minori.

## 11. Diritti e controllo dell'utente

L'utente può controllare i propri dati attraverso le impostazioni del dispositivo e le funzionalità disponibili nell'app.

In particolare, l'utente può:

- revocare i permessi Android concessi all'app;
- cancellare i dati dell'app dalle impostazioni Android;
- disinstallare l'app;
- evitare di configurare Last.fm;
- rimuovere o modificare le credenziali Last.fm, se la funzionalità è disponibile;
- non usare la funzione di condivisione del report.

## 12. Modifiche a questa informativa

Questa Privacy Policy può essere aggiornata per riflettere cambiamenti dell'app, delle funzionalità o dei requisiti normativi.

In caso di modifiche rilevanti, la data di efficacia sarà aggiornata.
